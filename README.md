# CVify AI

Create a professional CV in 60 seconds 🇿🇦

## Overview

CVify AI is a fast and easy-to-use web application that helps you generate a professional CV in just 60 seconds. Simply fill in your details, click "Generate CV", and download your CV as a PDF.

## Features

- ⚡ **Quick CV Generation** - Fill in your information and generate a professional CV instantly
- 📄 **PDF Download** - Download your CV as a PDF file with one click
- 🎨 **Professional Design** - Clean, modern, and responsive layout
- 📱 **Mobile Friendly** - Works seamlessly on desktop, tablet, and mobile devices
- 🇿🇦 **SA Focused** - Built for South African professionals

## How to Use

1. Enter your full name
2. Add your email address
3. Input your phone number
4. Enter your location
5. Write your professional summary
6. Click "Generate CV" to preview your CV
7. Click "Download PDF" to save your CV

## Technologies Used

- **HTML5** - Markup structure
- **CSS3** - Styling and responsive design
- **JavaScript** - Interactivity and PDF generation
- **html2pdf.js** - PDF generation library

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/bleeng157-commits/cvify-ai.git
```

2. Navigate to the project directory:
```bash
cd cvify-ai
```

3. Open `index.html` in your web browser

That's it! No installation or setup required.
